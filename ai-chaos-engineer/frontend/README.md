# AI Chaos Engineer - Frontend

React + Vite + Tailwind CSS UI for the AI Chaos Engineer.

## Setup

```bash
npm install
```

## Run

```bash
# Development mode
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Configuration

The frontend connects to the backend API. By default it uses `http://localhost:3001`.

To change this, create a `.env` file:

```env
VITE_API_URL=http://localhost:3001
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- PostCSS
